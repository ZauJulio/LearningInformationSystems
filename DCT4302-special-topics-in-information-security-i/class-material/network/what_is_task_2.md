## Network tools

- tracerout: Envia pacotes até um endereço:porta de destino usando o protocolo ICMP, mapeando cada salto até o fim do ttl.
- itracerout: Envia pacotes até um endereço:porta de destino usando o protocolo ICMP, retornando o host no ttl especificado no param.
- portscan: Verifica se uma porta qualquer está aberta, fechada ou filtrada.
- discovery: Retorna todos os endereços ip de uma rede especifica.
