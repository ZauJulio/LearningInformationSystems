# Caesar Decrypt

## Description

Try to decrypt a Caesar ciphered text.

## How was it done?

I created a function to find the frequency of characters, from that I calculated the distance from the most frequent letter for the letter 'e' and 'a', then I used these distances and some kicks later, I managed to find the key.

### Keys

- data/SgTAPSra -> -97
- data/3CWp9sI1 -> -83
- data/vevxaYL8 -> ? (I didn't find the key)
