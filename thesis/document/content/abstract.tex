\chapter*{Resumo}
\thispagestyle{empty}

No presente trabalho é descrito o desenvolvimento de um \textit{boilerplate} voltado à criação de aplicações e ferramentas interativas para análise e visualização de dados. O software desenvolvido oferece uma infraestrutura modular e extensível, integrando \texttt{GTK} para interface gráfica, \texttt{matplotlib} para visualização, \texttt{pandas} para manipulação de dados e algoritmos do \textit{scikit-learn} como exemplos de aprendizado de máquina aplicado. O sistema foi projetado com base em princípios de design orientado a objetos, priorizando encapsulamento e modularidade para reutilização de código. Entre os componentes desenvolvidos, destacam-se gerenciadores de estado, controladores de interface e um gerenciador genérico de modelos capaz de integrar diferentes algoritmos para processamento de dados. Este \textit{boilerplate} visa simplificar o desenvolvimento de ferramentas de análise de dados, oferecendo uma solução escalável e reutilizável para projetos acadêmicos e industriais.

\vspace{1.5ex}

Análise de Dados;

\chapter*{Abstract}
\thispagestyle{empty}

This paper describes the development of a boilerplate aimed at creating interactive applications and tools for data analysis and visualization. The developed software offers a modular and extensible infrastructure, integrating GTK for graphical interface, matplotlib for visualization, pandas for data manipulation and scikit-learn algorithms as examples of applied machine learning. The system was designed based on object-oriented design principles, prioritizing encapsulation and modularity for code reuse. Among the developed components, state managers, interface controllers and a generic model manager capable of integrating different algorithms for data processing stand out. This boilerplate aims to simplify the development of data analysis tools, offering a scalable and reusable solution for academic and industrial projects.

\vspace{1.5ex}

Data Visualization;
